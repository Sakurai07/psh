# psh
A shell for linux/windows written in python

# Linux:
Navigate to releases and download the .deb or binary

  **deb:**
  ```
  dpkg -i /path/to/.deb
  ```
  
  **binary**
  ```
  chmod +x /path/to/binary
  ./binary
  ```
